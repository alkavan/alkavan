# PROJECT AT

<img align="right" width="33%" src="project_at_11r_1024px.png" alt="PROJECT AT LOGO">

An umbrella project for software, protocols, licenses, and formats with embedded
theft-resistant mechanisms for digital asset and content distribution over
public networks.

### WG42
The working group designs and builds protocols and technologies for asset
distribution and storage in digitally abundant networked environments — with a
passion for freedom and authenticity.
