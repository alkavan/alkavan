# PROJECT AT

<img align="right" width="33%" src="project_at_11r_1024px.png" alt="PROJECT AT LOGO">

This is an umbrella project for software, protocols, formats, and licenses
with integrated authenticity and ownership of digital assets — which are intrinsically
designed against intellectual thievery.

A working group that designs and builds new protocols and technologies for authentic asset distribution
in digitally abundant environments — with a passion for freedom, decentralization, and ownership.
